# Software-Quiz

![Deployment Status](https://github.com/BAS-More/Software-Quiz/actions/workflows/jekyll-gh-pages.yml/badge.svg)

An interactive software engineering quiz application built with Jekyll and deployed via GitHub Pages.

## Features

- 10 software engineering questions covering various topics
- Interactive quiz interface with immediate feedback
- Score tracking and results display
- Responsive design for mobile and desktop
- Built with vanilla JavaScript and Jekyll

## Deployment

This site is automatically deployed to GitHub Pages using Jekyll when changes are pushed to the main branch.

**Live Site**: https://bas-more.github.io/Software-Quiz/

## Local Development

To run locally:

```bash
# Install Jekyll
gem install jekyll bundler

# Serve the site
jekyll serve
```

Then visit `http://localhost:4000` in your browser.

## Quiz Topics

The quiz covers:
- Object-Oriented Programming (OOP)
- Data Structures
- Algorithms and Complexity
- SOLID Principles
- Web Services and APIs
- Design Patterns
- Version Control
- Testing
- HTTP Methods
